Penjelasan Website

Website ini merupakan hasil dari skripsi yang berfokus pada identifikasi pneumonia menggunakan model Convolutional Neural Network (CNN) dengan arsitektur VGG19. Pada website ini, user (pengguna) dapat mengunggah gambar hasil pemeriksaan rontgen paru-paru untuk mengidentifikasi kemungkinan adanya pneumonia. Model CNN VGG19 telah dilatih sebelumnya menggunakan dataset yang mencakup gambar normal dan gambar dengan pneumonia. Hasil prediksi dari model ini akan ditampilkan kepada user dengan keterangan apakah gambar tersebut menunjukkan adanya pneumonia atau tidak, memberikan dukungan penting bagi diagnosis medis secara cepat dan akurat.
